# Лабораторная работа №1. Начало работы с MySQL и MySQL Workbench

**Задание**: 
Освоить базовые операции с СУБД MySQL через Workbench: создание БД, таблиц (`users`, `resume`), работа с типами данных, внешними ключами и каскадными действиями, проверка ограничений целостности.

**Результат**: 
Составлен конспект с SQL-запросами создания, наполнения и модификации таблиц, описанием ошибки внешнего ключа (вставка с несуществующим `userid`), а также подтверждением работы `CASCADE` при удалении и обновлении записей.

**Суть проделанной работы**:
Выполнены 10 заданий: создана БД `simpledb`, таблицы `users` (поля: `id`, `name`, `email`, `gender` ENUM, `bday`, `postal_code`, `rating`, `created` TIMESTAMP) и `resume` с внешним ключом на `users.id`. Данные добавлены вручную и SQL-запросами. Проверено, что вставка с `userid=9999` вызывает ошибку `1452`. Удаление пользователя удаляет его резюме, а изменение `id` пользователя корректно обновляет внешний ключ.
