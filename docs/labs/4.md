# Лабораторное занятие по реляционной алгебре. 
## Решение трёх задач на основе исходных отношений

**Задание:**  
1. Найти имена и телефоны клиентов, заказавших в одном заказе не менее двух игр с разными названиями.  
2. Найти табельный номер сотрудника, оформившего заказ, включающий игру с минимальным возрастом 12 лет.  
3. Найти фамилии и имена клиентов, купивших игры с минимальным числом игроков 2 и максимальным – 6.  

**Суть проделанной работы:**  
Выполнены операции реляционной алгебры: проекция, выборка, естественное и условное соединения, разность, переименование. Для каждой задачи поэтапно получены промежуточные отношения, выполнен анализ данных, исключены дубли и сформулированы итоговые ответы.
